Random items to know for the Dev Exam
  HTTP Codes
    200 - Request succeeded
    3xx - Redirection
    4xx - Client Error (ex: 404 - not found)
    5xx - Server Error

    Available
      Andriod, iOS, JavaScript (Browser)
      Java
      .Net
      Node.js
      PHP
      Python
      Ruby

      US-EAST-1

        some do not (Node.js)
